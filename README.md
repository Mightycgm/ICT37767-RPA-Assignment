# 🤖 ICT37767 Robotic Process Automation (RPA)
**Repository:** [ICT37767-RPA-Assignment](https://github.com/Mightycgm/ICT37767-RPA-Assignment)

Repository นี้จัดทำขึ้นเพื่อรวบรวมผลงาน แบบฝึกหัด (Assignments) และโปรเจกต์จบ (Final Project) ของรายวิชา ICT37767 ซึ่งเน้นไปที่การประยุกต์ใช้เทคโนโลยี RPA เพื่อเพิ่มประสิทธิภาพและลดระยะเวลาในการทำงานของกระบวนการทางธุรกิจ

---

## 📚 Assignments Overview
สรุปผลงานแบบฝึกหัดตลอดภาคเรียน:

* **Assignment 1:** 📧 **Automated Email System**
  * **Tool:** Make.com
  * **Detail:** การสร้างระบบ RPA สำหรับส่งอีเมลอัตโนมัติ
* **Assignment 2:** 🦄 **Web Data Extraction**
  * **Tool:** UiPath
  * **Detail:** การดึงข้อมูลชื่อยูนิคอร์นจากเว็บไซต์ rpasamples.com มาแสดงผลเป็น Alert Box
* **Assignment 3:** 📦 **Decision Logic (If/Else)**
  * **Tool:** UiPath
  * **Detail:** การทดลองใช้เงื่อนไข If/Else ในการตัดสินใจเลือกราคาการขนส่งสินค้าตามเกณฑ์ที่กำหนด
* **Assignment 4:** 📊 **Excel Integration**
  * **Tool:** UiPath
  * **Detail:** ต่อยอดจาก Assignment 2 โดยนำข้อมูลชื่อที่ดึงจากเว็บไซต์ มาจัดเก็บลงในไฟล์ Excel อย่างเป็นระบบ
* **Assignment 5 (Group Work):** 📁 **Email & File Management**
  * **Tool:** UiPath
  * **Detail:** การดึงไฟล์แนบจากอีเมลมาเก็บไว้ในระบบ -> คัดแยกและย้ายไฟล์ไปยังหมวดหมู่ที่ถูกต้อง -> เปลี่ยนชื่อไฟล์อัตโนมัติ

## 🚀 Final Project: Intern Registration Automation
**ชื่อโปรเจกต์:** การพัฒนาระบบบริหารจัดการผู้สมัครฝึกงานโดยกระบวนการหุ่นยนต์อัตโนมัติ

ระบบ RPA แบบ End-to-End ที่ออกแบบมาเพื่อแก้ไขปัญหา (Pain Points) ของฝ่ายทรัพยากรบุคคล (HR) ในการจัดการข้อมูลผู้สมัครฝึกงาน ลดการทำงานซ้ำซ้อน และเพิ่มความรวดเร็วในการแจ้งผลพิจารณาผ่านอีเมลอัตโนมัติ

### 🛠️ Architecture & Tech Stack
* **Frontend:** `HTML` (หน้าฟอร์มรับสมัคร)
* **Backend/Middleware:** `Google Apps Script` / `UiPath Studio`
* **Database:** `Google Sheets`
